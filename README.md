# 🌄 Uttarakhand Tourism Website

A beautiful and informative tourism site showcasing destinations across Uttarakhand, India.

## 🔥 Features
- Responsive design
- Popular destinations: Nainital, Mussoorie, Kedarnath, and more
- Image galleries
- Chat bot Support
- Weather API


## 🛠️ Tech Stack
- HTML, CSS, JavaScript
- GitHub Pages
- MONGO DB
- NODE.JS
- API
